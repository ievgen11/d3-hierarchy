## Overview

An example of how Redux + Immutable + Material + D3.js can work together to create a clean and robust application to display hierarchy data.

## Run locally

```sh
$ yarn run watch

```

## Tools used

-   React
-   Redux
-   Immutable.js
-   D3.js
-   RMWC
